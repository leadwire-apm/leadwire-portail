## Installation

### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/leadwire-apm/leadwire-ui.git

# go into app's directory
$ cd angular

# install app's dependencies
$ npm -g install grunt
$ npm -g install bower
$ npm install
$ bower install
```

## Usage

``` bash
# serve with hot reload at localhost:9001.
$ grunt serve

# build for production with minification /dist
$ grunt build
```
